# web相关

### 网页生成过程

1. `html`被html解析器生成DOM树。
2. `css`被解析器生成CSSOM树。
3. 结合DOM树和CSSOM树，生成一颗渲染树 `Render Tree`
4. 生成布局（flow），将所有渲染树的所有节点进行平面合成。
5. 将布局绘制在屏幕上。

### 重排：

当dom的变化影响了元素几何信息（dom对象的位置和尺寸大小），浏览器需重新计算元素的几何属性。
