`"D:\DProgramFiles\mysql\bin\mysqld.exe" --defaults-file="C:\ProgramData\MySQL\MySQL Server 5.7\my.ini" MySQL57`

`terminal=>mysql -u root -p`

`-p`即以用户密码方式登录。
